## Symbol System

| Symbol | Meaning |
|--------|---------|
| → | leads to, implies |
| ⇒ | transforms to |
| & | and, combine |
| \| | or, separator |
| » | sequence, then |
| ∴ | therefore |
| ∵ | because |
| ≈ | approximately |
| ✅ | done |
| ❌ | failed |
| ⚠️ | warning |

## Rules

- 30-50% token reduction target
- Skip articles, prepositions where unambiguous
- Use symbols over words
- Tables over prose
- Abbreviate: cfg, impl, perf, req, deps, val
- No explanations for obvious patterns
